# Flora Cosmica – Boutique de Flora Interestelar

E-commerce frontend desarrollado como proyecto del Módulo 2. Una boutique de plantas interestelares con diseño bioluminiscente y funcionalidad de carrito simulado.

---

## Demo

Abre `index.html` directamente en tu navegador (no requiere servidor).

---

## Estructura del proyecto

```
trabajo-modulo-2/
├── index.html               # Página de inicio (catálogo de productos)
├── css/
│   └── styles.css           # Estilos personalizados + variables CSS
├── js/
│   └── main.js              # Datos de productos, lógica del carrito, utilidades
└── pages/
    ├── detalle.html         # Página de detalle de producto
    ├── carrito.html         # Carrito de compras
    └── contacto.html        # Formulario de contacto
```

---

## Tecnologías

- **HTML5** semántico
- **Bootstrap 5.3** vía CDN (grid, navbar, utilities, badge)
- **CSS3** con variables custom, animaciones y diseño mobile-first
- **JavaScript** vanilla (DOM, eventos, arrays/objetos, localStorage)
- **Git/GitHub** para control de versiones
